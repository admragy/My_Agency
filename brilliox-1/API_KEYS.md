# Hunter Pro CRM - دليل المفاتيح المطلوبة

## مفاتيح قاعدة البيانات (اختر واحدة)

### الخيار 1: Replit PostgreSQL (مجاني - تلقائي)
- **DATABASE_URL** - يتم توفيره تلقائيًا من Replit
- لا تحتاج إعداد، يعمل مباشرة!

### الخيار 2: Supabase (مجاني/مدفوع)
| المفتاح | الوصف | من أين تحصل عليه |
|---------|--------|------------------|
| `SUPABASE_URL` | رابط مشروع Supabase | https://supabase.com → Settings → API |
| `SUPABASE_KEY` | مفتاح API | https://supabase.com → Settings → API → anon key |

## كيفية إضافة المفاتيح في Replit

1. اذهب إلى **Secrets** في القائمة الجانبية
2. اضغط **+ New Secret**
3. أدخل اسم المفتاح والقيمة
4. اضغط **Add Secret**
